This is a project that I have used for combining the tools I have made for myself for learning music,
and for learning about:

- Go
- HTMX
- templ
- authentication
- among other things

---

To build this program you will neet to have installed:

- Go
- templ
- sqlite and the sqlite driver from github.com/mattn/go-sqlite3

---

To build the app:

1. Generate the html files: `templ generate`
2. Compile the the app: `go build`
3. Start the app: `./musiikkiProjektit`
